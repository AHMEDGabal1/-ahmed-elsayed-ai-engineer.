## 🧠 Tech Stack

- **Framework**: Next.js 14 (App Router, TypeScript)
- **Styling**: Tailwind CSS (dark, premium UI, glassmorphism)
- **Animations**: Framer Motion (micro-interactions, scroll progress, hero motion)
- **Build**: Node.js + npm

## 🧩 Key Sections

- **Hero** – Bold, conversion-optimized entry with role typing, premium gradient visuals, and strong CTAs:
  - _View My Work_
  - _Work With Me_
  - _Download CV_
- **About** – Founder-style story focused on intelligent systems, LLMs, and automation.
- **Skills** – Modern skill cards segmented into AI/ML, Programming, and Frameworks/Tools with proficiency indicators.
- **Projects** – Business-focused AI projects in a **Problem → Solution → Result** format (ForsaHunt, Dentlyzer, microservices translation/summarization).
- **AI Agency Vision** – Visionary section about building an AI automation agency in the Middle East.
- **Experience & Education** – Clean, modern timeline using real CV data.
- **Contact** – High-conversion contact with direct links (email, LinkedIn, GitHub) and a floating “Start a Conversation” email CTA.

## 🚀 Getting Started (Local)

```bash
git clone https://github.com/AHMEDGabal1/-ahmed-elsayed-ai-engineer..git
cd -ahmed-elsayed-ai-engineer./website

npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

## 📦 Production Build

```bash
npm run build
npm start
```

You can host this on any Node-compatible platform (VPS, Render, Railway, etc.).
